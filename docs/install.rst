Installation
============

Requirements
------------

Python UT generator will work with Python >=3.6 or PyPy3.

Using PIP
----------------------

To install Python UT generator using `pip`_:

.. code-block:: bash

  $ pip install pip install pyutgenerator

To install from source:

.. code-block:: bash

  $ pip install -e .

.. _pip: https://pip.pypa.io/
